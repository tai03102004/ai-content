### Heading Hierarchy

- [ ] **One clear H1** as main theme (e.g., "Complete Guide to...")
- [ ] **3-5 major H2 sections** that support H1
- [ ] **Each H2 has 2-4 H3 subsections** for depth
- [ ] **H4 only when necessary** (avoid going too deep)
- [ ] **No conclusion section** in outline (added during writing)

### Logical Flow

- [ ] First section introduces the topic
- [ ] Middle sections dive into details
- [ ] Last section connects back to first (circular structure)
- [ ] Smooth transitions between sections
- [ ] Related topics grouped together

- [ ] Main content comprises **>80%** of total
- [ ] Supplemental content comprises **<20%** (FAQs, comparisons)
- [ ] Each section has **specific focus** (no overlap)

### Competitive Analysis

- [ ] Includes points **NOT found** in top 10 competitors
- [ ] **More comprehensive** than best competitor
- [ ] **Unique angle** or perspective
- [ ] **Better examples** and data

## ðŸ“– Article Methodology Requirements

For **EACH section**, must specify:

### 1. Content Format

Choose appropriate format:

    Paragraphs (for explanations, storytelling)
    Bullet points (for lists, features, benefits)
    Tables (for comparisons, specifications)
    Mixed (combination of above)

### 2. Estimated Word Count

Guidelines:

    Introduction: 150-250 words
    Main H2 sections: 300-500 words each
    H3 subsections: 150-300 words each
    Supplemental sections: 100-200 words each

### 3. Main Ideas to Cover

List specific points, such as:

    Definition and explanation
    Why it matters (benefits)
    How it works (process)
    Common mistakes to avoid
    Best practices
    Real-world examples

### 4. Examples, Data, Evidence

Include:

    Statistics (with sources and year)
    Case studies (real brands/projects)
    Expert quotes (with credentials)
    Screenshots or visuals
    Before/after comparisons

### 5. Section Connections

Explain how section relates to:

    Previous section (transition from...)
    Next section (leads into...)
    Overall article theme (supports main point by...)
